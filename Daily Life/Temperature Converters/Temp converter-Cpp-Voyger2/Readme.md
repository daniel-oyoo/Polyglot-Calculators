# Project 
A basic temperature converter made in C++ which converts Fahrenheit to Celsius and Vice Versa

## Implementations
- Using of arithmetic operators to convert the temperatures
- Easy to understand and Compact

## Usage
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius

## Format
- Enter 'C' for Fahrenheit to Celsius
- Enter 'F' for Celsius to Fahrenheit

## Requirements
- Any C++ Compiler works
