# Project 
A basic temperature converter made in C++ which converts Fahrenheit to Celsius and Vice Versa

## Implementations
- Using of arithmetic operators to convert the temperatures
- Easy to understand and Compact

## Usage
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius

## Format
- Enter 'C' for Converting to Celsius
- Enter 'F' for Converting to Fahrenheit

## Requirements
- Any C++ Compiler works

### Compile the Code
Navigate to the directory containing `main.cpp` and use the C++ compiler (`g++`) to build the executable:
```
g++ main.cpp -o temp_converter
./temp_converter
```
